Welcome to our wiki. Please use the navigation bar on the left side of your screen to quickly navigate to any section.

How to generate a key pair

1. In order to generate your keys, select the "Generate Keys" option from the navigation bar.




2. Once you are on that page, click on the "Generate Keys" button.



3. Your browser will prompt you to save the file somewhere. Navigate to the folder you would like to save the key to, and then save it.



4. Extract the "keys.zip" file that you just downloaded.



5. Your "public_key.pem" file and your "private_key.pem" files should now be available to you. NEVER share your private key with anyone, under any circumstances. If someone gains access to your private key they will permanently be able to decrypt and view all of your records.

Creating a new record (adding patient data to the blockchain)

1. Click on the "Add Record" button in the navigation bar.



2. Click the "Browse..." button.

3. Navigate to the patient's "public_key.pem" file and select it.

4. Click the "Upload" button.

You should now see the "Key" form be filled out, similar to the image below.


5. Fill out each of the remaining fields.

6. Click the "Add block" button.

7. You should now see a page similar to this.

8. Click "Download updated key" to overwrite the updated "public_key.pem" file on your system with the new transaction number.

9. Alternatively, you can only decide to manually note down the transaction ID. It will be available to you at the bottom of the Key form, similar to the image provided.
