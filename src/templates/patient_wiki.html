How to use our website (patient)

This website will be a wiki on all the features of this website, how to use these features and future goals we may have to imporve our site. This is a PHR (personal health record) system that uses a blockchain to store patients data. We have multiple navigation bar that send you to each feature page. We will go into detail on how to use each feature.
